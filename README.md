# Library_management_system.

1. Introduction

A Library Management System is a software application used to manage books in a library. It helps in storing information about books such as book ID, title, and author.
In traditional libraries, all records are maintained manually, which is time-consuming and error-prone. This system solves that problem by using a computer-based approach.
This project is developed using C++ programming language and follows Object-Oriented Pr

2. Objectives of the Project
The main objectives of this project are:
•	To manage library records efficiently 
•	To reduce manual work and errors 
•	To store book details in a structured way 
•	To provide easy operations like: 
   o	Adding a book 
   o	Searching a book 
   o	Issuing a book 
   o	Returning a book 
•	To understand and implement OOP concepts in C++

3. Methodology / System Design
The system is designed using Object-Oriented Programming concepts.
* Class Design:
•	A class named Book is created 
•	It contains: 
  o	Data Members: 
    	Book ID 
    	Title 
    	Author 
    	Status (Issued/Available) 
  o	Member Functions: 
    	Add Book 
    	Display Book 
    	Search Book 
    	Issue Book 
      Return Book

* Working Process:
1.	User selects an option from the menu 
2.	System performs the selected operation 
3.	Data is stored in objects (array of books) 
4.	Output is displayed on the screen
   
* System Flow:
•	Start Program 
•	Show Menu 
•	Perform Operation 
•	Repeat until Exit

4. Implementation / Source Code
The project is implemented in C++ using classes and objects.
* Key Features of Code:
•	Uses class Book 
•	Uses array to store multiple books 
•	Menu-driven program 
•	Functions for each operation

5. Conclusion
The Library Management System successfully demonstrates how basic library operations can be managed using C++ and OOP concepts.
It reduces manual work, improves accuracy, and provides a simple interface for users. This project also helps in understanding real-world applications of programming.
